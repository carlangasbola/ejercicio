Cambio de contraseña

Instrucciones para el cambio de contraseña

La contraseña debe tener una longitud de 8 a 32 caracteres debe incluir, por lo menos, tres caracteres de los siguientes:

  • Letra mayúscula del alfabeto (A-Z)
  • Letra minúscula del alfabeto (a-z)
  • Cifra decimal (0-9)
  • Caracteres especiales, tales como < # } { * + - [, etc

No utilices la letra “ñ” ni letras con tilde (por ejemplo: è, ü, etc.) ni otros caracteres especiales, ya que pueden causar errores. En este caso, intenta otra combinación para la contraseña.
